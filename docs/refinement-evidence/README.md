# Refinement Evidence Folder

Store before/after visual proof for each refinement note ticket.

## Folder format

- `docs/refinement-evidence/RN-###/before-01.png`
- `docs/refinement-evidence/RN-###/after-01.png`

Optional:
- `before-02.png`, `after-02.png`
- short videos (`.mp4`) when interaction is important

## Minimum closure evidence

To close a ticket:
1. At least one `before` image
2. At least one `after` image
3. Ticket must be referenced in `REFINEMENT_NOTES_BOARD.md`
